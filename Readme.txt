Restful Web Services are available in the Backend folder
Client-Web Services are present in the Frontend folder

1. Have the FrontEnd and BackEnd in a single Main folder.
2. Open with an IDE (VSCode, WebStorm)
3. Open 2 Terminals 
4. One Terminal is assigned to go to the FrontEnd Directory. 
	// cd Frontend
5. One Terminal is assigned to go to the BackEnd Directory.
   	 // cd Backend
6. Run "npm install" or "npm i" through both the terminals to install all dependencies.
7. Type "npm start" in the BackEnd directory to run the server.
8. Type "npm start" in the FrontEnd directory to run the WebClient
